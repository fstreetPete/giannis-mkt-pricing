Combined expected return (sum of bets)
Each bet pays $1 per contract if it wins. EV($) = q·B·(1−p)/p·(1−vig) − (1−q)·B, where p = entry price, q = your win prob, B = bet size. "Boston No" wins unless he goes to Boston, so q = 1 − P(Boston). Same logic for Miami No. Kelly weight is the growth-optimal way to split a pot across the three, solved jointly so it accounts for how the bets overlap — it's a relative ratio, independent of your bet sizes and bankroll. A bet can drop to a small weight (or 0) if it's redundant with a better-priced one, and a slightly −EV bet can earn weight if it hedges the others.
